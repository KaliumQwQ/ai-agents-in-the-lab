01

AI coding agents, from scratch

No prior experience assumed. What is this thing, why is it different from autocomplete, and what does it actually touch?

The core idea

An agent is a loop, not a chatbot

A chatbot answers. An agent is given a goal and runs a loop — the same one you run at the bench debugging an experiment.

Read

your project, a stack trace, a 10,000-row data file.

Act

write a script, run it in a real shell, edit many files.

Observe

read the result, see the error, try again — on its own.

Why it matters

Autocomplete vs. an agent

Most people's mental model of "AI coding" is the grey ghost-text in an editor. That's a different, much smaller thing.

🔡 Autocomplete old

  • Finishes the line you're typing
  • Never leaves the editor
  • Can't run, test, or check anything
  • No memory of your project as a whole
  • You drive every single step

🤖 Agent now

  • Takes a goal, plans the steps
  • Edits files, runs scripts, reads errors
  • Searches the web, your data, your docs
  • Holds the whole repo in context
  • You review; it does the legwork

Skills — teaching the agent a craft

Reusable folders of expertise the agent loads on demand. This is where a generic model becomes your group's specialist.

Definition

What is a “skill”?

A skill is a small folder — instructions, a few scripts, some examples — that the agent pulls in only when the task calls for it. Write it once; it works in every future conversation.

  • Turns a general agent into a specialist on demand
  • Write once → reuse across every project and teammate
  • An open standard — portable across different tools
  • Invoked like a command: /code-review, /research …
  • Loaded lazily, so it never clutters the context until needed

Everything that makes this annoying by hand:

  • a junk first line — gzipped metadata, unreadable by eye
  • tab-separated, scientific notation
  • cycle markers glued onto the time column (34.44000 1 CYCLE) — breaks naïve parsers
  • 9,999 rows across 8 cycles to slice apart
Tedious to parse, trivial to describe. That gap is exactly what the agent closes.
The agent's script

analyze_cv.py — written from one sentence

import numpy as np

def load_cv(path):
    """E (V), i (A/cm²), t (s) from a CHI / CS-Studio export."""
    E, i, t = [], [], []
    for ln in path.read_text(encoding="utf-8-sig").splitlines()[2:]:
        p = ln.split("\t")
        if len(p) >= 3:                         # skip blanks / markers
            E.append(float(p[0])); i.append(float(p[1]))
            t.append(float(p[2].split()[0]))    # drop trailing "N CYCLE"
    return map(np.array, (E, i, t))

def peaks(E, i):
    rising = np.gradient(E) > 0                  # forward = anodic sweep
    fwd, rev = np.where(rising)[0], np.where(~rising)[0]
    return fwd[np.argmax(i[fwd])], rev[np.argmin(i[rev])]

E, i, t = load_cv(Path("ferrocene_CV.txt"))
pa, pc  = peaks(E, i)
dEp     = (E[pa] - E[pc]) * 1000                 # peak separation / mV
Ehalf   = (E[pa] + E[pc]) / 2                    # formal potential / V

Handles the junk header, the cycle markers, and the forward/reverse split — the exact things that trip up a quick hand-written parser.

Reading it

What the numbers mean

E½ = ½(Epa+Epc) = 0.40 V — the formal potential of the Fc/Fc⁺ couple, its fingerprint.
ΔEp = 343 mV ≫ 59 mV — far from the reversible ideal: quasi-reversible, the classic signature of uncompensated resistance / sluggish kinetics.
|ipa/ipc| = 1.45 — both directions present; some asymmetry in the return peak.

The agent didn't fake a clean 59 mV — it reported what the data says and named the likely cause. A real, slightly imperfect curve is more instructive, and being honest about it is the bar for trusting analysis.

04

The research itself

Now to the science these tools are in service of — a proposal about what a Tafel slope really contains.

The project

Rethinking OER Tafel slopes from Au–O bond polarity

Directly measure the electrosorption valency γ — the partial charge on a surface oxo species on gold — and use it to take apart the Tafel slope.

γ

electrons exchanged per adsorbed O. A basic descriptor of metal–oxygen bonding — yet never measured directly.

OAT

chemical oxygen-atom transfer to a molecule reads γ out cleanly, with no kinetic background.

Tafel

the slope everyone reads as mechanism — finally split into its true, separable parts.

The problem

The Tafel slope is read as a clean fingerprint — it isn't

Standard practice maps a slope (60, 120 mV/dec) onto a rate-determining electron-transfer step, and reads off α and a mechanism. That assumes the oxo intermediate is a fixed, integer-electron species. But the charge stored in the adsorbate layer is two things at once:

\[ \frac{dq_\text{ads}}{dE} \;=\; F\,\underbrace{\gamma\,\frac{d\Gamma}{dE}}_{\text{coverage}} \;+\; F\,\underbrace{\Gamma\,\frac{d\gamma}{dE}}_{\text{Au–O bond polarity}} \]

Electron transfer

genuine PCET kinetics — what we think we're measuring.

Coverage

the number of surface oxygens changing with E.

Bond polarity

each oxygen's charge re-tuning with E — silently absorbed.

If γ varies with potential, part of the “Tafel slope” is adsorbate-layer charging — not pure kinetics. The α and mechanism you read off are systematically biased.
Prior art

Conway & Liu got close — and got stuck

  • Potential-relaxation on oxidized Pt & Ni: read intermediates from the open-circuit decay after the current is cut
  • Extracted an apparent valency y ≈ 0.23 and 0.37 on two distinct Pt regions
  • The slope relation \( b_t^{-1} = b_T^{-1} - b_c^{-1} \) was verified experimentally
  • Two regions appear around 1.83 V, where the Tafel slope itself changes
But the same slope can come from small γ, lateral interactions, or non-Langmuir coverage. Relaxation data alone cannot separate them — Conway said so himself.

A minimum in the adsorption pseudo-capacitance Cφ(E) marks two kinds of surface-oxygen behaviour — the clue this proposal turns into a number.

Conway & Liu, Proc. R. Soc. A 429, 375 (1990); Faraday Trans. I 83, 1063 (1987).

The slope algebra

Why the measured valency looks small

Conway's relaxation law plus a Langmuir isotherm turn the capacitance slope into an apparent valency:

\[ C\,\frac{dE}{dt} = -\,i(E) \qquad \frac{1}{b_t}=\frac{1}{b_T}-\frac{1}{b_c} \]
\[ \frac{\theta}{1-\theta}=K\,e^{\gamma F E/RT}\;\Rightarrow\; C_\phi \propto e^{\gamma F E/RT} \]
\[ \frac{dE}{d\log_{10}C_\phi}=\frac{2.303\,RT}{\gamma F}=\frac{59\ \text{mV}}{\gamma} \]
A perfect one-electron adsorbate gives 59 mV/decade. Conway measured 258 and 161 mV → γ ≈ 0.23 and 0.37.
59 mV/γ alone can't tell a genuinely small γ apart from Frumkin lateral interactions or non-Langmuir coverage. The valency is only apparent — the gap this proposal closes.
The key idea

γ is the Au–O bond polarity

\[ \gamma \;=\; \frac{1}{F}\left(\frac{\partial q_M}{\partial \Gamma}\right)_{E} \]

γ measures how much electron density gold hands to oxygen — i.e. how ionic vs. covalent the Au–O bond is. That charge separation is an interfacial dipole, so γ is simultaneously an electrical quantity and a measure of the oxygen's chemical reactivity.

γ → 0 γ ≈ 1 γ → 2
covalent oxyl
weakly polarized, electrophilic O — an oxidant
partial charge
the regime Conway's data lives in
ionic oxide
fully reduced, nucleophilic O²⁻
Crucially γ changes with potential (dγ/dE ≠ 0): every surface oxygen's electrophilicity is re-tuned as E moves — a different thing from how many oxygens there are.
The read-out

Chemical OAT carries γ across the interface

Oadsδ− + PPh3 → O=PPh3 — the PPh₃ acceptor approaches, abstracts the oxygen, and carries it off as closed-shell O²⁻ — taking exactly the partial charge γ with it. OAT's electrical fingerprint is γ itself, nothing else mixed in.
Why aprotic

Switch off PCET, isolate γ

In water, oxygen can leave by two routes carrying different charge. The trick is to close one of them.

(A) PCET reduction — needs water / protons

Oads + 2H+ + 2e− → H2O  ·  n = 2, drags in the full Faradaic kinetics (the contamination Conway was stuck with)

✕

(B) Chemical OAT — aprotic solvent

Oads + PPh3 → O=PPh3  ·  n = γ, no proton source needed

No protons ⇒ pathway (A) is off. Oxygen can only leave by (B), so the only electrical observable becomes γ — with zero Faradaic-kinetics background. Aprotic isn't a compromise; it's the cleanest possible configuration.

The intuition

No wire out — so how does it discharge?

The external circuit is open — an ammeter reads zero. Yet the potential still moves. Why?

  • Two capacitors in parallel: the double layer Cdl and the adsorption pseudocapacitance Cφ (the Au–O charge γFΓ).
  • In parallel with them, a chemical-reaction resistance R — the OAT pathway that consumes the oxo.
  • External open ⇒ the capacitors can only discharge through R → E decays. No wire needed.
Conway's self-discharge, driven by chemistry: (Cdl+Cφ) dE/dt = −irxn.
The working equation

Open circuit + charge conservation

At open circuit no net external current flows. Split the metal charge into double-layer + bound-oxo charge, then set dqext = 0 at constant Cdl:

\[ q_M = C_\text{dl}\,(E-E_\text{pzc}) + \gamma F\,\Gamma_{\mathrm O} \quad\xrightarrow{\;dq_\text{ext}=0\;}\quad 0 = C_\text{dl}\,dE + \gamma F\,d\Gamma_{\mathrm O} \]
\[ \boxed{\;\dfrac{dE}{d\Gamma_{\mathrm O}} \;=\; -\,\dfrac{\gamma F}{C_\text{dl}}\;} \]

E(t)

from the open-circuit-potential transient.

ΓO(t)

from in-situ SERS, absolutely calibrated by NMR.

Cdl

from impedance spectroscopy.

Eliminate time between E(t) and Γ(t) → take the slope → γ, with no kinetic model at all. That independence is the whole advantage over Conway.

Experimental design

Three independent observables, one number

🔬 in-situ SERS

Au–O band intensity → continuous ΓO(t); ¹⁸O labelling confirms the band assignment.

⏱️ OCP transient

E(t) over decades of time — now driven by chemical OAT, not self-discharge.

🧪 ³¹P NMR + 2 acceptors

Quantifies O=PPh₃ → absolute coverage; PPh₃ vs. styrene probe one vs. two oxygen species.

The payoff

Splitting the Tafel slope, for real

With γ(E) measured independently, the observed OER Tafel slope decomposes into three honest contributions instead of one assumed kinetic number:

electron transfer
coverage
Au–O polarity

Electron transfer

the genuine PCET rate-determining step and transfer coefficient α.

Coverage

how the number of surface oxygens shifts with potential.

Bond polarity

dγ/dE — each oxygen's electrophilicity being re-tuned with potential.

The last two were always silently lumped into “electron transfer”. Now they're separable — a corrected baseline for reading any OER Tafel slope.

Why it matters

A corrected baseline for OER mechanism

Turns a slope-inferred guess into a directly measured, product-calibrated number for γ.
Cleanly separates the electron-transfer (PCET) contribution from the partial-charge (desorption) one — the split Conway wanted but couldn't make.
Gives a model-free reference against which DFT and microkinetic claims can be checked.
The bond-polarity term dγ/dE ties an electrochemical observable directly to the electrophilicity of the OER intermediate — a quantitative bridge between Tafel analysis and molecular metal–oxo chemistry, where the same dipole governs selective oxidation.
